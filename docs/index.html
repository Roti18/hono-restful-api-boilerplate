Overview

This API provides a secure, high-performance interface for managing resources. Built with Hono, it adheres to strict RESTful principles and security best practices.

Base URL: http://localhost:3000/api

Authentication

Authenticate requests using either an API Key or Bearer Token.

API Key

X-API-Key: your_api_key

Bearer Token

Authorization: Bearer your_access_token

Items Resource

GET /items

List all items with pagination support.

curl -X GET http://localhost:3000/api/items \
  -H "X-API-Key: your_key"
POST /items

Create a new item.

curl -X POST http://localhost:3000/api/items \
  -H "Content-Type: application/json" \
  -d '{"name": "New Item", "description": "Desc"}'

Auth Resource

POST /auth/token

Exchange credentials for an access token.

curl -X POST http://localhost:3000/api/auth/token \
  -d 'grant_type=client_credentials&client_id=...&client_secret=...'
